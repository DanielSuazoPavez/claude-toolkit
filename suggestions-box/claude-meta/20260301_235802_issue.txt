Revisit activation knowledge scoring in evaluate-skill rubric. Currently D1 (Knowledge Delta) treats activation knowledge as lower value ('keep sparingly'), but the simplify skill analysis showed that targeted activation checklists ARE the value for procedural/command-style skills. 'Claude already knows what N+1 is' vs 'Claude checks for N+1 right now on this diff' is a meaningful behavioral difference. Suggestion: acknowledge that activation knowledge is genuinely valuable when it's targeted and justified — a curated 'check these 16 specific things' list is expert curation even if each individual item isn't novel. The expertise is in the selection and combination, not in the individual concepts. This ties into the broader command-vs-skill type discussion — command-style resources get unfairly penalized on D1 because their value is activation, not novelty.
